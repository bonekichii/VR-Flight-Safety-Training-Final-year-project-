# How to Run Your StereoKit Safety Demo

**Your Professional VR Project is Ready.**

## 1. Open in Visual Studio
1.  Navigate to: `C:\Users\Vega\.gemini\antigravity\scratch\SafetyDemo\`
2.  Double-click **`SafetyDemo.csproj`**.
3.  This will open **Visual Studio 2022**.

## 2. Run the Project
1.  In Visual Studio, look at the top toolbar.
2.  Make sure the "Green Play Button" says **SafetyDemo**.
3.  Click **Play** (or press `F5`).

## 3. Controls (Simulator)
Since you are likely testing on a PC first:
*   **Mouse:** Move the "Hand" cursor.
*   **Right-Click + Drag:** Rotate the camera.
*   **Scroll Wheel:** Move the hand forward/backward.
*   **Left Click:** "Pinch" (Select/Click buttons).

## 4. Deploy to Quest (Later)
StereoKit makes this easy. When you are ready to put it on the headset, just change the "Solution Platform" from `x64` to `ARM64` and plug in your Quest.
